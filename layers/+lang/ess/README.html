1. Description

This layer adds support for statistical programming languages to Spacemacs.

1.1. Features:

  • Syntax highlighting
  • Auto-completion
  • Syntax-checking via lintr
  • Additional data viewer for R via ess-R-data-view
  • Support for Org-Babel
  • Showing of inline help for R constructs
  • Repl support via R terminal
  • Support for S, SAS and R
  • Much more via the ESS Project

2. Install

2.1. Layer

To use this configuration layer, add it to your ~/.spacemacs. You will need to add ess to the existing dotspacemacs-configuration-layers list in this file.

2.2. Linting

Install lintr library via the R terminal to enable syntax checking. To do so start the R terminal and type below code. 

install.packages("lintr")

2.3. LSP

This backend uses an external server to provide the various IDE integrations and a more modern UI integration in spacemacs.

It requires installing the external server via: 

install.packages('languageserver')

Enable the lsp layer to activate this backend as well as setting the layer variable ess-r-backend: 

(ess :variables ess-r-backend 'lsp)

3. Options

Use a key binding to insert <-: 

(setq-default dotspacemacs-configuration-layers '((ess :variables
                                                       ess-assign-key "\M--")))
